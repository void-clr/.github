---

**void-clr** is a development organization focused on building small, sharp and reliable tools around the C# and .NET ecosystem.

The name comes from two ideas:

- **`void`** — no noise, no bloat, only what matters.
- **`clr`** — the Common Language Runtime, the execution layer behind .NET.

We build with a simple principle:

> keep the surface minimal, keep the internals solid.

## Focus

- C# / .NET libraries
- Runtime-aware tooling
- Backend and infrastructure components
- Developer experience utilities
- Experiments around CLR, MSIL and compilation

## Philosophy

```txt
minimal APIs
predictable behavior
clean abstractions
runtime-first thinking
```

We prefer simple tools that do one thing well, clear code over clever code, and systems that stay understandable as they grow.

## Projects

This organization is designed to host:

- reusable .NET packages;
- CLI tools for developers;
- backend-oriented utilities;
- low-level experiments around the CLR;
- clean, minimal open-source projects.
